## 📞 Quick Reference

### Important Commands:
```bash
# Build debug
./gradlew assembleDebug

# Build release APK
./gradlew assembleRelease

# Build release AAB
./gradlew bundleRelease

# Install on device
adb install app-release.apk

# View logs
adb logcat | grep CodeCore
```

### Important Files:
```
google-services.json → app/
local.properties → project root
keystore.properties → project root (create later)
codecore-release.keystore → project root (create later)
```
